Создай современный лендинг для продукта "CollabCode" — AI-native платформы для парного программирования.

## О продукте
CollabCode — это расширение для VS Code и веб-приложение, которое заменяет Live Share, добавляя:
- ИИ-ассистента, который участвует в сессии как третий разработчик
- Голосовые аннотации к коду
- Призрачные курсоры с предсказанием действий
- Асинхронные сессии с ИИ-доигровкой стиля напарника
- Instant Replay с возможностью форка кода из любой точки истории

## Требования к дизайну
- Стиль: футуристичный, тёмная тема с неоновыми акцентами (фиолетовый + циан)
- Эффекты: стекло (glassmorphism), плавные градиенты, микроанимации при скролле
- Шрифты: моноширинный для кода, геометрический гротеск для заголовков
- Адаптив: под мобильные, планшеты и десктоп

## Структура секций
1. Hero — заголовок "CollabCode: AI-native pair programming". Подзаголовок: "Код вместе. Даже когда вы не вместе". Две CTA: "Попробовать бесплатно" и "Смотреть демо" (модалка с видео).
2. Проблема vs Решение — слева: разорванные сессии, конфликты, отсутствие асинхронности. Справа: Nexus с фичами.
3. Ключевые фичи — карточки с иконками и краткими описаниями (AI Pilot, Ghost Cursors, Voice Annotations, Async Sessions, Instant Replay). В каждой карточке при наведении появляется гифка-демонстрация.
4. Как это работает — пошаговая схема: 1) Установил расширение → 2) Создал сессию → 3) Позвал напарника ссылкой → 4) ИИ присоединяется автоматически.
5. Сравнение с аналогами — таблица: Nexus vs Live Share vs Code With Me. По строкам: кол-во участников, асинхронный режим, AI-участие, голосовые аннотации, replay. Везде у Nexus галочки.
6. Социальное доказательство — вымышленные цитаты (Senior Dev, Team Lead, OSS Maintainer) + счётчик "Уже 5000+ команд".
7. CTA + подвал — форма с e-mail для раннего доступа, ссылки на соцсети и GitHub.
